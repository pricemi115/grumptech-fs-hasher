BatchHashEntry

FSBatchHashEntryModule. BatchHashEntry

(private) new BatchHashEntry()

Source:
  • fsBatchHashEntry.mjs, line 41
  • Handles hashing of a list of file system sources. A "batch" item is simply a list of files/directories where there is no expected heirarichal relationship. In many ways, other than building, this class acts very much like a directory.
