API Wrapper for the Report Method

Throws:
  • thrown if this instance is no longer registered.
Type
ReferenceError
Returns:
  • A promise that when resolved will provide a string containing a CSV delimited report. If the heirarchy has not been built or the root source is busy, the promise will resolve to an empty string. CSV Format: Field #1: Type - '(D)'-Directory, '(F)'-File, '(B)'-Batch Field #2: Source - Path of the File System object. Will be prepended with spaces corresponding to the depth of the result item. Field #3: Digest - Message digest of the item. If it has not been computed, it will default to undefined.
